## xhxStatuBar
一个简单易用的状态栏指示器

## 显示成功信息
```objc
 [XHXStatusBarHUD showSuccess:@"加载成功!"];
```
## 显示失败信息
```objc
[XHXStatusBarHUD showError:@"加载失败!"];
```
## 显示加载信息
```objc
[XHXStatusBarHUD showLoading:@"正在加载中..."];
```
## 显示普通信息
```objc
[XHXStatusBarHUD showMessage:@"没有什么事!!!!" image:nil];
```
## 隐藏信息
```objc
[XHXStatusBarHUD hide];
```
